# Portal de Exámenes

Proyecto final realizado en grupo, desarrollado como un **portal de exámenes básico**.

## Descripción
Este proyecto corresponde a un **trabajo final**, realizado junto a **dos compañeros**, con el objetivo de aplicar conocimientos de desarrollo web y backend.

El sistema implementa un portal simple para la gestión y realización de exámenes, utilizando una arquitectura separada entre frontend y backend.

## Tecnologías Utilizadas
- Angular (TypeScript)
- Java + Spring Boot
- JWT (autenticación sin estado)
- MySQL
- API REST (JSON)

## Ejecución
1. Ejecutar el backend con Spring Boot
2. Configurar la base de datos
3. Ejecutar el frontend Angular
4. Acceder desde el navegador
